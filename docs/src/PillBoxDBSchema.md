### **taboo_case**

비트마스크 기법을 이용하여 처리

코드 순서는 의약품안전사용서비스(DUR) 순서를 따른다

적합성 검사는 병용 금기, 특정연령금기, 임부금기, 노인금기만 다룬다.

* 병용 금기 : A = 0x01
* 특정연령 금기 : B = 0x02
* 임부 금기 : C = 0x04
* 용량 금기 : D = 0x08
* 투여기간 금기 : E = 0x10
* 노인금기 : F = 0x20
* 효능군 중복 : G = 0x30
* 분할주의 : H = 0x40
* 첨가제주의 : I = 0x50
* 당뇨금기 : J = 0x60 (DUR 이외)
* 혈압 : K = 0x70 (DUR 이외)
